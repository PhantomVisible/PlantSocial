# Gamification Service

This service handles the gamification aspects of PlantSocial, such as points, badges, and achievements.

## Prerequisites
- Java 21 (or compatible version)
- Gradle

## How to Start

1. Open a terminal in the `gamification-service` directory.
2. Run the application using Gradle:
   ```bash
   ./gradlew bootRun
   ```
   (On Windows use `gradlew.bat bootRun`)

The service will start on its configured port (typically `8081`).
