# PlantSocial Frontend

This is the Angular web application for PlantSocial.

## Prerequisites
- Node.js (v18 or higher recommended)
- npm

## Installation

Before starting for the first time, install the dependencies:
```bash
npm install
```

## How to Start

1. Open a terminal in the `frontend` directory.
2. Start the Angular development server:
   ```bash
   npm start
   ```
   Or using the Angular CLI directly:
   ```bash
   ng serve
   ```

The application will be available at `http://localhost:4200/`. Ensure that the backend and API gateway services are running so the frontend can retrieve data.
